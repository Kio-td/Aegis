Welcome

The Welcome module allows you to have custom Welcome and Leave messages on your server.

How to setup welcome messages

  1. Mark the channel you'd like welcome and leave messages to go in by doing !setchannel channelname
    • For example !setchannel joins
  1. Set the type of message you'd like to be used by doing !settype type
    • For example !settype embed
    • There are 3 types; Embed, Text, and DM


Configuring the welcome message

Message Functions

Description

$title

First line of text on the message

$description

Following text field below

$thumbnail

The square icon in the top right

$pic

The banner below the welcome

$field

No clue lol

%user%

The users name will go here automatically

%guild%

The guild name will go here automatically


To set the welcome message the command is !setwelcome new Configuration

To view your current welcome message at anytime just do !setwelcome current

Here's an example of setting up a welcome message;

!setwelcome new $title:        Welcome to %guild%, %user%
$description:        Welcome to %guild%, Please read the #rules and be nice! Feel free to invite your friends! And do `!!verify`
$thumbnail:      https://i.imgur.com/rGjTwG6.jpg
$pic: https://i.imgur.com/CshqQHa.png

Configuring the leaving message

To set the leave message the command is !setleave new Configuration

You can use the same functions above to configure your leave message.

To view your current leave message at anytime just do !setleave current
